# Релизы

История пользовательских и технических изменений сайта "Волшебная живая баня".

## v1.0.0 - 2026-06-24

Первый оформленный production-релиз нового сайта.

### Добавлено

- Современный адаптивный одностраничный сайт для бани в Калуге.
- SEO-оптимизация: meta-теги, Open Graph, Twitter Card, canonical, geo-разметка.
- Schema.org JSON-LD для локального бизнеса, рейтинга, адреса, карты и каталога услуг.
- Расширенный блок услуг по банной тематике: аренда, чан, купель, парение, веники, мангал, чай, программы отдыха.
- Большая фотогалерея с модальным просмотром, миниатюрами, стрелками и свайпами.
- Плавающий аудиоплеер с треками, прогрессом, громкостью, раскрытием и компактным мини-режимом.
- Круглый музыкальный акцент на первом экране.
- Ссылки на Яндекс Карты во всех адресных местах сайта.
- Кнопки связи: телефон и MAX.
- Мобильная шапка с кнопками телефона, карты и меню.
- Favicon, PWA-иконки и `site.webmanifest`.
- `robots.txt` и `sitemap.xml` с изображениями.
- GitHub Pages workflow для публикации статического сайта.

### Улучшено

- Сетка первого экрана и информационного блока с адресом.
- Мобильная адаптация, включая порядок кнопок в шапке.
- Визуальная стилизация аудиоплеера под тематику бани.
- Компактность мини-плеера: название текущего трека, шкала прогресса, большая кнопка play/pause, вертикальная громкость.
- Галерея стала удобнее для просмотра на телефоне.

### Контакты и интеграции

- Основной номер: `+7 910 510-53-48`.
- MAX: `https://web.max.ru/413130154`.
- Яндекс Карты: `https://yandex.ru/maps/org/volshebnaya_zhivaya_banya/110090709629/`.

## Как выпускать следующий релиз

1. Обновить этот файл: добавить новую версию сверху.
2. Проверить сайт локально и убедиться, что основные сценарии работают.
3. Сделать коммит на русском языке.
4. Запушить `main`.
5. Поставить git-тег:

```bash
git tag -a v1.0.1 -m "Релиз v1.0.1"
git push origin v1.0.1
```

Если нужен полноценный GitHub Release с описанием в интерфейсе GitHub, его можно создать из тега и текста текущего файла.
