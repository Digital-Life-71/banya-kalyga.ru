# Инструкции для Codex

- После каждой правки сайта делать `git add`, коммит на русском и `git push origin main`.
- Не записывать логины, пароли, токены, cookie и другие секреты в файлы репозитория, включая `AGENTS.md`.
- Доступы к сайту брать только из локального файла `.env.local` или из временного сообщения пользователя в текущем чате.
- Локальный файл `.env.local` не коммитить. Ожидаемые переменные:
  - `SITE_ADMIN_URL`
  - `SITE_ADMIN_LOGIN`
  - `SITE_ADMIN_PASSWORD`
- Если `.env.local` отсутствует, перед работой с админкой попросить доступы у пользователя и не сохранять их в git.
